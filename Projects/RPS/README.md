# Rock-Paper-Scissors Game

A simple command-line implementation of the classic Rock-Paper-Scissors game for two players.

## How to Play

1. Run the Python script in your terminal or command prompt.
2. Enter the number of rounds you want to play.
3. For each round:
   - Player 1 enters their choice (rock, paper, or scissor)
   - Player 2 enters their choice (rock, paper, or scissor)
4. The game will determine the winner of each round based on standard Rock-Paper-Scissors rules:
   - Rock beats Scissors
   - Scissors beat Paper
   - Paper beats Rock
5. After all rounds are completed, the final scores are displayed and the overall winner is announced.

## Features

- Supports any number of rounds
- Tracks scores for both players
- Handles invalid inputs by repeating the round
- Displays final results with winner declaration

## Requirements

- Python 3.x
